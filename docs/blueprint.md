# **App Name**: Visit Uzbekistan

## Core Features:

- Tour Catalog: Display a catalog of available tours with photos, descriptions, itineraries, and prices.
- Individual Tour Builder: Allow users to submit custom tour requests.
- Team Profiles: Showcase the team members, including leaders, directors, guides, and drivers, with their photos and information.
- Image Gallery: Display a gallery of photos from past tours.
- Contact Form: Enable users to submit inquiries and requests via a contact form.
- Tour Search: A comprehensive search tool that returns the best Uzbekistan tours for the user, based on keywords.

## Style Guidelines:

- Primary color: Light gray (#F2F2F2) to maintain the clean and airy aesthetic, resembling the provided screenshot.
- Background color: White (#FFFFFF) for a crisp, clean look.
- Accent color: A darker shade of gray (#A9A9A9) for buttons and interactive elements, ensuring they stand out.
- Font: 'Inter', sans-serif, is a neutral and modern typeface suitable for both headlines and body text.
- Follow the layout and spacing as shown in the provided screenshot, maintaining a consistent and clean design.
- Use simple, line-based icons to complement the clean design aesthetic.
- Subtle, fast transitions for button clicks to give an immediate sense of response to user actions.
